## 表格練習
---
| 表格1 | 表格2 |
| ----- | ----- |
| 內容1 | 內容3 |
| 內容2 | 內容4 |

---


| Name  | Quantity | Comment |
|:-----:|:--------:|:-------:|
| Apple |    3     |  Japan  |
|  Egg  |    12    | Vietnam |

---
| 指令     | 描述         | 快速鍵     |
| -------- | ------------ | ---------- |
| 向左移動 | 向左移動焦點 | Ctrl-Left  |
| 向右移動 | 向右移動焦點 | Ctrl-Right |
| 向上移動 | 向上移動焦點 | Ctrl-Up    |

---
| 指令       | 描述                 | 快速鍵     |
| ---------- | -------------------- | ---------- |
| 下一單元格 | 移動到下一個單元格   | Tab        |
| 前一單元格 | 移動到前一個單元格   | Shift-Tab  |
| 下一列     | 移動到下一列         | Enter      |
| 跳出       | 從表格編輯模式中跳出 | Ctrl-Enter |
